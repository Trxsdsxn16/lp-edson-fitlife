# LP-EB1-FitLifeConnect-Edson

Landing Page responsive para FitLife Connect, plataforma que conecta a personas, entrenadores personales, nutricionistas y gimnasios en un solo lugar.

## Tecnologías utilizadas
HTML5, CSS3, JavaScript (validación de formulario, menú móvil). Sin frameworks ni librerías externas.

## Justificación del diseño

**Paleta de colores:** fondo claro (blanco / verde salvia pálido) con un acento distinto por segmento de usuario — coral para personas, azul para entrenadores, morado para nutricionistas, dorado para gimnasios. Esto refuerza la arquitectura de la información: el color ayuda a reconocer de inmediato a qué segmento pertenece cada sección, y da suficiente contraste para cumplir accesibilidad (AA) en botones y texto.

**Tipografía:** Archivo Black para títulos (peso e impacto, transmite energía/deporte) combinada con Inter para el cuerpo de texto (alta legibilidad en pantallas pequeñas, clave para diseño inclusivo).

**Disposición visual:** el hero abre con un selector rápido por segmento (idea central del caso: cuatro tipos de usuario, un solo lugar). Cada sección alterna el fondo y el orden del contenido para dar ritmo visual sin perder consistencia. El formulario de contacto cierra el recorrido, después de que el usuario ya conoce los cuatro servicios.

## Cómo ejecutar el proyecto localmente
1. Descarga o clona la carpeta completa (manteniendo `css/` y `js/` junto a `index.html`).
2. Abre `index.html` con doble clic, o usa la extensión "Live Server" de VS Code.
3. Para publicarlo en la web: sube la carpeta a GitHub y activa GitHub Pages en Settings → Pages.
